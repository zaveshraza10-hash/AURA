# AURA ATM - AI-Powered Intelligent ATM System

## Overview
A next-generation ATM system with facial recognition, QR login, instant loan approval, voice commands, and advanced AI features.

## Features
- Face Recognition Login
- QR Code Card-less Access
- Instant Loan Eligibility with AI
- Voice Command Navigation
- Fraud Detection
- Financial Health Dashboard

## Setup

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python app.py`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## AI Models
- Face Recognition: FaceNet
- Loan Eligibility: Random Forest
- Fraud Detection: Isolation Forest

## Tech Stack
- Frontend: React, Material-UI
- Backend: FastAPI, Python
- AI: OpenCV, Scikit-learn
- Database: SQLite/PostgreSQL
